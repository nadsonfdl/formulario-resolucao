📌 Guia de Resolução

Inclua sempre:

  • 🧠 Intenção do cliente
  • 📝 Descrição do problema
  • 🛠️ Ações realizadas (com caminhos no sistema)
  • ⚙️ Configurações alteradas
  • 📊 Acesso ao banco (tabela, coluna, query utilizada)
  • 📋 Resultado final e retorno do cliente
  • 📌 Ações pendentes e follow-up (se necessário)

Exemplo:

Cliente relatou erro ao imprimir orçamento manual. Acessado: Prontuário > PRE-OP > Gerar Orçamento Manual. Ajustado banco: tabela unidadesconfig, campo LIBERAR_ORCAMENTO_MANUAL = "S". No sistema: Config > Filial > aba Margens. Selecionadas opções de impressão. Problema resolvido.
